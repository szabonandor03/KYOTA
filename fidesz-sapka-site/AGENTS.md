# FIDESZ SAPKA Site Bootstrap

This directory uses the one-prompt fast path unless the task is clearly broader than a bounded site edit.

## Before Editing

1. Read `./CLAUDE.md`.
2. Read `../kyota-wiki/NOW.md`.
3. For routine site work, follow `../kyota-wiki/schema/fidesz_sapka_single_prompt_workflow.md`.

## Site Rules

- Preserve the current archive / shared-Drive leak posture.
- Do not introduce hype language, album material, or polished hidden videos unless the rollout plan changes.
- Keep diffs small when multiple UI choices are plausible.
- Use existing assets before introducing new ones.

## Verify

- Run `npm run build` in `fidesz-sapka-site/` for normal site changes.
